# クラウドコース端末後片付け（2026年1月以降）

## Google Chrome

- GitHubにアクセスし訓練生アカウントからサインアウト

- 訓練生個人アカウントを「Chromeからログアウト」

- 「Chromeプロファイルの管理」ですべてのプロファイルを削除（新しいプロファイルが作成される）

- Ctrl＋Shift＋Deleteで閲覧履歴データを削除

- AWSマネジメントコンソールのrootおよびIAMのユーザーID入力履歴を削除

- ハローワーク求人検索の求職者番号の履歴を削除

## Visual Studio Code

- 残しておく拡張機能：Japanese Language Pack、Error Lenz、HTMLHint、Live Server、Prettier（上記以外はアンインストール）

- アカウントのメニューからGitHubをサインアウトする

## Cursor

- 訓練生のアカウントをサインアウト

## Kiro

- 訓練生のアカウントをサインアウト

## RLogin

- 登録されているサーバー接続設定をすべて削除

## pgAdmin 4

- 登録されているサーバー接続設定をすべて削除

## Typing Land

- 設定画面でユーザーを削除

## Word/Excel/PowerPoint

- 「ファイル」タブをクリック→「開く」→「最近使った項目」の一覧が表示されるので、いずれかのファイル名を右クリックして「お気に入り以外の項目をクリア」

## 不要なアプリフォルダおよびファイル削除

- デスクトップに「Clear.bat」ファイルを作成してメモ帳で開き、下記スクリプトを貼り付けて保存し、ファイルをダブルクリックで実行する

```DOS
@echo off
chcp 65001 >nul
echo ===============================================
echo クラウドコース端末 - 不要ファイル削除スクリプト
echo ===============================================
echo 下記処理を実施します
echo.
echo - Cursor、Kiro、Antigravity 初期化
echo.
echo - Git設定ファイル削除
echo.
echo - ユーザーファイル削除（ドキュメント、ピクチャー、ムービー、ミュージックおよびダウンロードフォルダ）
echo.
echo - AWS/SSH設定フォルダの削除
echo.
echo - Windows履歴の削除
echo.
echo 中止する場合はここでCtrl+Cを押してください。
pause

echo.
echo [1/7] Cursorの初期化...
if exist "%AppData%\Cursor" (
    rmdir /s /q "%AppData%\Cursor"
    echo   Cursor AppDataフォルダを削除しました
) else (
    echo   Cursor AppDataフォルダは存在しませんでした
)
if exist "%USERPROFILE%\.cursor" (
    rmdir /s /q "%USERPROFILE%\.cursor"
    echo   Cursor ホームフォルダを削除しました
) else (
    echo   Cursor ホームフォルダは存在しませんでした
)

echo.
echo [2/7] Kiroの初期化...
if exist "%AppData%\Kiro" (
    rmdir /s /q "%AppData%\Kiro"
    echo   Kiro AppDataフォルダを削除しました
) else (
    echo   Kiro AppDataフォルダは存在しませんでした
)
if exist "%USERPROFILE%\.kiro" (
    rmdir /s /q "%USERPROFILE%\.kiro"
    echo   Kiro ホームフォルダを削除しました
) else (
    echo   Kiro ホームフォルダは存在しませんでした
)

echo.
echo [3/7] Antigravityの初期化...
if exist "%AppData%\Antigravity" (
    rmdir /s /q "%AppData%\Antigravity"
    echo   Antigravity AppDataフォルダを削除しました
) else (
    echo   Antigravity AppDataフォルダは存在しませんでした
)
if exist "%USERPROFILE%\.antigravity" (
    rmdir /s /q "%USERPROFILE%\.antigravity"
    echo   Antigravity ホームフォルダを削除しました
) else (
    echo   Antigravity ホームフォルダは存在しませんでした
)

echo.
echo [4/7] Git設定ファイルの削除...
if exist "%USERPROFILE%\.gitconfig" (
    del /q "%USERPROFILE%\.gitconfig"
    echo   .gitconfigを削除しました
) else (
    echo   .gitconfigは存在しませんでした
)

echo.
echo [5/7] ユーザーフォルダ内のファイル削除...
if exist "%USERPROFILE%\Documents" (
    del /q "%USERPROFILE%\Documents\*" 2>nul
    echo   ドキュメントフォルダをクリアしました
)
if exist "%USERPROFILE%\Pictures" (
    del /q "%USERPROFILE%\Pictures\Screenshots\*" 2>nul
    del /q "%USERPROFILE%\Pictures\*" 2>nul
    echo   ピクチャーフォルダをクリアしました
)
if exist "%USERPROFILE%\Videos" (
    del /q "%USERPROFILE%\Videos\Captures\*" 2>nul
    del /q "%USERPROFILE%\Videos\*" 2>nul
    echo   ビデオフォルダをクリアしました
)
if exist "%USERPROFILE%\Music" (
    del /q "%USERPROFILE%\Music\*" 2>nul
    echo   ミュージックフォルダをクリアしました
)
if exist "%USERPROFILE%\Downloads" (
    del /q "%USERPROFILE%\Downloads\*" 2>nul
    echo   ダウンロードフォルダをクリアしました
)

echo.
echo [6/7] AWS/SSH設定フォルダの削除...
if exist "%USERPROFILE%\.aws" (
    rmdir /s /q "%USERPROFILE%\.aws"
    echo   .awsフォルダを削除しました
) else (
    echo   .awsフォルダは存在しませんでした
)
if exist "%USERPROFILE%\.ssh" (
    rmdir /s /q "%USERPROFILE%\.ssh"
    echo   .sshフォルダを削除しました
) else (
    echo   .sshフォルダは存在しませんでした
)

echo.
echo [7/7] Windows履歴の削除...
if exist "%AppData%\Microsoft\Windows\Recent" (
    del /q "%AppData%\Microsoft\Windows\Recent\*" 2>nul
    echo   最近使ったファイル履歴を削除しました
)
if exist "%AppData%\Microsoft\Windows\Recent\AutomaticDestinations" (
    del /q "%AppData%\Microsoft\Windows\Recent\AutomaticDestinations\*" 2>nul
    echo   クイックアクセス履歴を削除しました
)
if exist "%AppData%\Microsoft\Windows\Recent\CustomDestinations" (
    del /q "%AppData%\Microsoft\Windows\Recent\CustomDestinations\*" 2>nul
    echo   ピン留めアクセス履歴を削除しました
)

echo.
echo =============================================
echo 不要ファイルおよびフォルダの削除作業が完了しました
echo デスクトップ上およびゴミ箱の中のファイルは手動で
echo 削除してください
echo =============================================
echo.
pause
```

## Windows設定

- AWSから取得した、.pem、.ppk、.csvといったセキュリティ情報のファイルが残って入れば削除

- 資格情報マネージャーから、pgAdminおよびgit関連の資格情報を削除

- Officeを利用するのに訓練生自身のマイクロソフトアカウントを使っている場合はアカウント設定を解除（設定→アカウント→アカウント）

- メモ帳を開いてすべてのタブを閉じる

- 既定のアプリ設定からファイルの種類に対するアプリ関連付けを確認
  
  - .txt：メモ帳
  
  - .html：Crome

- 画面がダークモードの場合は通常のモードに戻す

- Display設定：125%

- デスクトップ上に残すもの：ごみ箱、PC、ダウンロードフォルダ、訓練生フォルダ、Rlogin、DB Browser (SQLite)
  
  - 上記以外のデスクトップ上に作成されたフォルダやファイルを削除

- タスクバー上に残すもの：ペイント、Explorer、Chrome、SnippingTool、メモ帳、VSCode、Cursor、Kiro、Marktext、pgAdmin、Draw.io、Typing Land
  
  - 上記以外のタスクバーに登録されたショートカットを削除

- 最後にゴミ箱を空にする
