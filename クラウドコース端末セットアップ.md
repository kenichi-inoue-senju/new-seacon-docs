[Google Chrome - Google の高速で安全なブラウザをダウンロード](https://www.google.com/intl/ja/chrome/)

## Marktext

[Release 0.17.1 · marktext/marktext · GitHub](https://github.com/marktext/marktext/releases/tag/v0.17.1)

- marktext-setup.exeをダウンロード（ローカルにセットアップファイルを置かないインストールできない場合がある。）

- インストール後、Marktextを実行しメニューからThemaをDarkにする。

- Windows設定で、規定のアプリ設定画面下の方にある「ファイルの種類で規定値を選択する」から「.md」拡張子をMarktextに関連付ける。

## ClassScreenShare設定

- Windows Firewall：管理者権限でターミナルあるいはコマンドプロンプトを開いて下記コマンド実行（WebRTC用）
  
  ```powershell
  netsh advfirewall firewall add rule name="ClassScreenShare-WebRTC-Out" dir=out action=allow protocol=UDP remoteport=49152-65535
  ```

- Edgeのアドレスバーに「edge://flags」を入力し、下記セキュリティ設定変更後Edge再起動：
  
  -  Anonymize local IPs exposed by WebRTC
     （edge://flags/#enable-webrtc-hide-local-ips-with-mdns）
    →「Disabled」に変更
  
  - Insecure origins treated as secure
    （edge://flags/#unsafely-treat-insecure-origin-as-secure）
    → 「http://a1001202205:3000」を追加して「有効」に変更

- ショートカットファイルのプロパティを開き、「ショートカット」タブ下部の信頼されていないショートカットを「許可」とする
