## pwd

print working directory

`pwd`

```
/home/
├── ec2-user/  <= 今ここ
└── ssm-user/ 
```

自分がいる場所が `/home/ec2-user` ディレクトリであることを確認しました。

---

## ls

list

`ls /home`

```
/home/
├── ec2-user/
└── ssm-user/
```

`/home` の中にあるユーザー用ディレクトリ `ec2-user` と `ssm-user` を一覧で表示しました。

---

`ls /`

```
/
├── bin/
├── boot/
├── dev/
├── etc/
├── home/
：
├── tmp/
├── usr/
└── var/
```

ルートディレクトリ（いちばん上の階層 `/` ）にどのようなディレクトリがあるかを一覧で見ました。

---

`ls -a`

```
/home/ec2-user/
├── ./
├── ../
├── .bash_hostory
├── .bash_logout
├── .bash_profile
├── .bashrc
├── .lesshst
└── .ssh/
```

先頭が `.` の隠しファイルも含めて、カレントディレクトリにあるすべてのファイルやディレクトリを表示しました。

---

## cd

change dirctory

`cd /home`

```
/
└── home/  <= 今ここ
```

現在の場所を `/home` ディレクトリに移動しました。`pwd`コマンドで確認してみましょう。

---

`cd ec2-user`

```
/
└── home/
    └─ ec2-user/  <= 今ここ
```

`/home` の中の `ec2-user` ディレクトリにさらに移動しました。`pwd`コマンドで確認してみましょう。

---

`cd ..`

```
/
└── home/  <= 今ここ
```

一つ上の階層（親ディレクトリ）である `/home` に戻りました。`pwd`コマンドで確認してみましょう。

---

`cd`

```
/
└── home/
    └─ ec2-user/  <= 今ここ
```

ユーザーのホームディレクトリ（`/home/ec2-user`）に戻りました。`pwd`コマンドで確認してみましょう。

---

## touch

`touch test1.txt`

```
/home/ec2-user/
└── test1.txt  <= 作成された
```

`/home/ec2-user` の中に空のファイル `test1.txt` を新しく作成しました。`ls`コマンドで確認してみましょう。

---

## mkdir

make directory

`mkdir doc`

```
/home/ec2-user/
├── doc/  <= 作成された
└── test1.txt
```

`/home/ec2-user` の中に `doc` という名前の新しいディレクトリを作成しました。`ls`コマンドで確認してみましょう。

---

`mkdir doc2 doc3`

```
/home/ec2-user/
├── doc/
├── doc2/  <= 作成された
├── doc3/  <= 作成された
└── test1.txt
```

`doc2` と `doc3` という二つのディレクトリを一度に作成しました。`ls`コマンドで確認してみましょう。

---

## rmdir

remove directory

`rmdir doc3`

```
/home/ec2-user/
├── doc/
├── doc2/
└── test1.txt
```

中身が何も入っていないディレクトリ `doc3` を削除しました。`ls`コマンドで確認してみましょう。

---

## cp

copy

`cp test1.txt test2.txt`

```
/home/ec2-user/
├── doc/
├── doc2/
├── test1.txt
└── test2.txt  <= 複製された
```

`test1.txt` を元に同じ場所に `test2.txt` というコピーを作成しました。`ls`コマンドで確認してみましょう。

---

`cp test1.txt test2.txt doc`

```
/home/ec2-user/
├── doc/
│   ├─ test1.txt  <= 複製された
│   └─ test2.txt  <= 複製された
├── doc2/
├── test1.txt
└── test2.txt
```

カレントディレクトリにある `test1.txt` と `test2.txt` を、`doc` ディレクトリの中にコピーしました。`ls doc`コマンドで確認してみましょう。

---

`cp doc/test2.txt test3.txt`

```
/home/ec2-user/
├── doc/
│   ├─ test1.txt
│   └─ test2.txt
├── doc2/
├── test1.txt
├── test2.txt
└── test3.txt  <= 複製された
```

`doc` ディレクトリの中にある `test2.txt` を元に、現在の場所に `test3.txt` というコピーを作成しました。`ls`コマンドで確認してみましょう。

---

## mv

move

`mv test3.txt test4.txt`

```
/home/ec2-user/
├── doc/
│   ├─ test1.txt
│   └─ test2.txt
├── doc2/
├── test1.txt
├── test2.txt
└── test4.txt  <= 名前変更された
```

`test3.txt` というファイルの名前を `test4.txt` に変更しました。`ls`コマンドで確認してみましょう。

---

`mv test1.txt test2.txt doc2`

```
/home/ec2-user/
├── doc/
│   ├─ test1.txt
│   └─ test2.txt
├── doc2/
│   ├─ test1.txt  <= 移動された
│   └─ test2.txt  <= 移動された
└── test4.txt
```

現在の場所にある `test1.txt` と `test2.txt` を、`doc2` ディレクトリの中へ移動しました。`ls`および`ls doc`コマンドで確認してみましょう。

---

`mv doc doc1`

```
/home/ec2-user/
├── doc1/  <= 名前変更された
│   ├─ test1.txt
│   └─ test2.txt
├── doc2/
│   ├─ test1.txt
│   └─ test2.txt
└── test4.txt
```

ディレクトリ名 `doc` を `doc1` に変更しました。`ls`コマンドで確認してみましょう。

---

`mv doc1 doc2`

```
/home/ec2-user/
├── doc2/
│   ├── doc1/  <= 移動された
│   │   ├─ test1.txt
│   │   └─ test2.txt
│   ├─ test1.txt
│   └─ test2.txt
└── test4.txt
```

`doc1` ディレクトリを `doc2` ディレクトリの中に移動しました。`ls`、`ls doc2`および`ls doc2/doc1`コマンドで確認してみましょう。

---

`mv doc2/test1.txt .`

```
/home/ec2-user/
├── doc2/
│   ├── doc1/
│   │   ├─ test1.txt
│   │   └─ test2.txt
│   └─ test2.txt
├── test1.txt  <= 移動された
└── test4.txt
```

`doc2` ディレクトリの中にある `test1.txt` を、今いる場所（カレントディレクトリ）に移動しました。`ls`コマンドで確認してみましょう。

---

## rm

remove

`rm test4.txt`

```
/home/ec2-user/
├── doc2/
│   ├── doc1/
│   │   ├─ test1.txt
│   │   └─ test2.txt
│   └─ test2.txt
└── test1.txt
```

ファイル `test4.txt` を削除しました。`ls`コマンドで確認してみましょう。

---

`rm -r doc2`

```
/home/ec2-user/
└── test1.txt
```

`doc2` ディレクトリを中身ごとすべて削除しました。`ls`コマンドで確認してみましょう。
